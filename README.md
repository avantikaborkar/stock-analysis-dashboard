Stock Analysis Dashboard (First Year Project)

This is a simple stock analysis and backtesting project I built during my 1st year of engineering.
The goal of this project is to understand how basic trading strategies work using historical stock data and visualize them using Python.

Features
- Load and analyze stock data from CSV files  
- Calculate Simple Moving Averages (SMA 5 and SMA 10)  
- Visualize stock price trends using graphs  
- Basic backtesting of buy/sell signals  
- Identify simple trading opportunities using moving averages  

Tech Stack
- Python  
- Pandas  
- Plotly (for visualization)

What the Project Does

This project takes historical stock price data and:
- Calculates trend indicators (like SMA 5 and SMA 10)
- Plots stock price movement over time
- Generates simple buy/sell signals based on crossover strategy
- Helps understand how basic algorithmic trading ideas work
